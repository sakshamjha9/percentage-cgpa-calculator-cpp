# Percentage & CGPA Calculator

A simple C++ project that calculates:

- Percentage
- CGPA
- Grade Points
- Academic Performance

## Features

- Multiple subject support
- Credit-based CGPA calculation
- Automatic grade point assignment
- Performance evaluation

## Technologies Used

- C++
- Arrays
- Loops
- Conditional Statements
- iomanip

## Grade Point Mapping

| Marks | Grade Point |
|--------|------------|
| 90+ | 10 |
| 75-89 | 9 |
| 65-74 | 8 |
| 55-64 | 7 |
| 50-54 | 6 |
| 45-49 | 5 |
| 40-44 | 4 |
| Below 40 | 0 |
